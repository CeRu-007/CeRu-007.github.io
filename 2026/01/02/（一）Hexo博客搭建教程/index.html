前言

Hexo 是一个快速、简洁且高效的博客框架。Hexo 使用 Markdown（或其他渲染引擎）解析文章，在几秒内，即可利用靓丽的主题生成静态网页。

本教程将带你从零开始搭建一个 Hexo 博客。

Hexo 基于 Node.js，所以首先需要安装 Node.js。

Windows 用户：

  • 访问 Node.js 官网
  • 下载 LTS 版本（长期支持版）
  • 运行安装程序，一路”下一步”即可

常见问题

Q: 如何更换端口？

A: 使用 hexo server -p 4001 可以指定端口。

Q: 如何新建页面？

A: 使用 hexo new page "about" 可以创建新页面。

Q: 如何安装主题？

A: 下载主题到 themes 目录，然后在 _config.yml 中设置 theme: 主题名称。

Q: 如何删除文章？

A: 直接删除 source/_posts/ 下的对应文件，然后重新生成。
